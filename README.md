# Daraz Clone

A Flutter e-commerce app with Daraz-style product listing.

## Features
- Login & Profile screens
- Product listing with categories (All, Electronics, Jewelery, Men's Clothing)
- Auto-sliding banner
- Search functionality
- Pull-to-refresh

## Architecture
- Single vertical scroll using CustomScrollView
- TabBarView with NeverScrollableScrollPhysics for horizontal swipe
- Provider for state management

## Run
```bash
cd task
flutter pub get
flutter run
```
